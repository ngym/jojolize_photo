# jojolize_photo
画像2から人物を切り出しつつ透過させて画像1に合成して「ゴゴゴゴゴ」とかをジョジョ風に付けるコード。
ChatGPTが提案してくれたコードを適当に編集した。

フォントはこちらのを使わせていただきました。
http://www2.gol.com/users/glow/jojo_font.html

## tmp3.py
スタンドにしたい人物の写った画像から、人物を抜き出す

## merge.py
抜き出したスタンド画像を、スタンド使いの画像に合成する

## gogogo.py
効果音「ゴゴゴゴゴゴ」を付加。
キーバインドが特殊なので注意。

## dododo.py
効果音「ドドドドドド」を付加。
キーバインドが特殊なので注意。

## ban.py
効果音「バーン」を付加。
キーバインドが特殊なので注意。
